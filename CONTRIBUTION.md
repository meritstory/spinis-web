# Contributing to this repository

## General

To start a new project based on this repository, you must follow these steps:
- Fork this repository
- Clone your forked repository

To contribute to this repository, you must follow these steps:
- Clone this repository or use your fresh forked repository
- Create a new branch for your feature or bug fix
- Make your changes
- Commit and push your changes
- Create a pull request
